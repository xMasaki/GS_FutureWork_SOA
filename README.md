## Objetivo do Projeto
O objetivo principal do projeto é a criação uma plataforma corporativa de cursos que prepare a empresa e seus colaboradores para o futuro, acompanhando o ritmo acelerado das transformações tecnológicas por meio de upskilling e reskilling.

A nossa proposta utiliza ferramentas para melhorar a experiência de trabalho e ampliar as oportunidades de crescimento, fortalecendo uma cultura corporativa mais justa, inclusiva e sustentável.

A plataforma orienta cada colaborador recomendando cursos de acordo com a função dela na empresa, garantindo que todos estejam preparados para novas funções, novos mercados e até carreiras que ainda irão surgir, impulsionando a capacidade da empresa de inovar e competir de forma estratégica no longo prazo.

## Funcionalidade
O sistema é uma API REST desenvolvida em Spring Boot que gerencia usuários e cursos, permitindo criar, listar, atualizar e excluir registros.

Os usuários possuem informações pessoais e endereço, enquanto os cursos têm título, duração, ano, categoria e professor, ambos mapeados para tabelas no MySQL.

A estrutura possui separação entre entidades, DTOs, repositórios e controllers. O sistema permite atualizações específicas, valida dados antes de persistir, e organiza respostas em páginas.
